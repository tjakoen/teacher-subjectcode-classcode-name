# Module 1 - Introduction

> Sample course-material unit. On GitHub this lives on a branch named
> `m1-intro`; publishing copies this folder into each student repo's
> `content/m1-intro/`, where it renders in the repo file view.

## What this is

Course material is just **Markdown + assets**. You can:

- write lessons in Markdown (this file),
- drop **images** next to it and embed them inline (`![](./diagram.png)`),
- attach **PDFs** - they open in GitHub's built-in viewer,
- link out to **YouTube** for video.

## Example lesson content

Welcome to the course. This module covers the setup and the ground rules.

Read this in your own repo under `content/m1-intro/`. New material will appear
here as the course progresses - it never touches your `notes/`, `journal/`, or
`project/` folders.
